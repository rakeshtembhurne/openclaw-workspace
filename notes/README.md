# Notes Directory

This directory contains all project notes, research, and documentation.

## Note Types

- **Project notes**: `<project-name>-notes.md`
- **Research**: `<topic>-research.md`
- **Planning**: `<project-name>-plan.md`
- **Meetings**: `<date>-meeting.md`

## Organizing Notes

Create subdirectories by topic or project:

```
notes/
├── projects/         # Project-specific notes
├── research/         # Research documents
├── ideas/            # Ideas and brainstorming
└── archive/          # Old notes (no longer active)
```

## Note Template

```markdown
# Title

**Date:** YYYY-MM-DD
**Status:** Active/Archived
**Tags:** tag1, tag2

## Summary
Brief description of what this note covers.

## Details
Main content goes here.

## Action Items
- [ ] Task 1
- [ ] Task 2

## Related
- Link to related projects
- Link to related notes
```
